# [Leetcoding] CheatSheet

## Array Big O

| Method type | Big O       |
| ----------- | ----------- |
| push        | O(1)        |
| pop         | O(1)        |
| shift       | O(n)        |
| unshift     | O(n)        |
| concat      | O(n)        |
| slice       | O(n)        |
| splice      | O(n)        |
| sort        | O(n\*log n) |
| map         | O(n)        |

| Method type    | Big O |
| -------------- | ----- |
| Insertion      | O(1)  |
| Removal        | O(1)  |
| Searching      | O(n)  |
| Access         | O(1)  |
|                |       |
| Object.keys    | O(n)  |
| Object.values  | O(n)  |
| Object.entries | O(n)  |
| hasOwnProperty | O(1)  |
